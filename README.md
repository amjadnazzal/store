# نظام إدارة متجر الملابس - Clothing Store Management System

برنامج سطح مكتب شامل لإدارة متاجر الملابس، مطور باستخدام C# و Windows Forms مع قاعدة بيانات SQLite.

## الميزات الرئيسية

### 1. إدارة المنتجات (المخزون)
- إضافة وتعديل وحذف المنتجات
- تتبع المخزون والكميات المتاحة
- دعم الباركود لكل منتج
- إدارة معلومات المنتج (الاسم، الوصف، المقاس، اللون، الأسعار)
- ربط المنتجات بالموردين
- تحميل صور للمنتجات
- البحث السريع في المنتجات
- تنبيهات للمنتجات منخفضة المخزون

### 2. نقطة البيع (POS)
- واجهة بيع سهلة ومتطورة
- مسح الباركود لإضافة المنتجات
- حساب المجاميع والخصومات تلقائياً
- دعم أنواع مختلفة من الخصم (نسبة مئوية أو مبلغ ثابت)
- اختيار طريقة الدفع (نقدي أو بطاقة)
- حساب الباقي للعمليات النقدية
- طباعة الفواتير
- ربط المبيعات بالعملاء

### 3. إدارة العملاء
- تسجيل بيانات العملاء الدائمين
- البحث السريع بالاسم أو رقم الهاتف
- تتبع سجل مشتريات العملاء
- ربط الفواتير بالعملاء

### 4. إدارة الموردين
- إدارة بيانات الموردين
- تتبع المنتجات حسب المورد
- معلومات الاتصال الكاملة

### 5. النسخ الاحتياطي والاستعادة
- إنشاء نسخ احتياطية من قاعدة البيانات
- استعادة البيانات من النسخ الاحتياطية
- حماية البيانات من الفقدان

### 6. التقارير والإحصائيات (قيد التطوير)
- تقارير المبيعات اليومية والشهرية
- تقارير الأرباح
- تقارير المخزون
- المنتجات الأكثر مبيعاً
- تقارير المنتجات منخفضة المخزون

## متطلبات النظام

- Windows 10 أو أحدث
- .NET 6.0 Runtime
- مساحة قرص صلب: 100 ميجابايت على الأقل
- ذاكرة وصول عشوائي: 2 جيجابايت على الأقل
- قارئ باركود USB (اختياري)
- طابعة لطباعة الفواتير (اختياري)

## التثبيت والتشغيل

### 1. تثبيت .NET 6.0 Runtime
- قم بتحميل وتثبيت .NET 6.0 Runtime من الموقع الرسمي لـ Microsoft
- [رابط التحميل](https://dotnet.microsoft.com/download/dotnet/6.0)

### 2. بناء المشروع
```bash
# استنساخ المشروع أو تحميل الملفات
git clone [repository-url]

# الانتقال إلى مجلد المشروع
cd ClothingStoreManager

# بناء المشروع
dotnet build

# تشغيل التطبيق
dotnet run
```

### 3. التشغيل المباشر
- قم بفتح ملف `ClothingStoreManager.sln` في Visual Studio
- اضغط F5 لتشغيل التطبيق

## كيفية الاستخدام

### البدء السريع

1. **إضافة الموردين**
   - انتقل إلى "المخزون" > "إدارة الموردين"
   - أضف موردين جدد بالمعلومات الأساسية

2. **إضافة المنتجات**
   - انتقل إلى "المخزون" > "إدارة المنتجات"
   - أضف منتجات جديدة مع الباركود والأسعار

3. **إضافة العملاء**
   - انتقل إلى "المبيعات" > "إدارة العملاء"
   - أضف عملاء دائمين للمتجر

4. **بدء البيع**
   - انتقل إلى "المبيعات" > "نقطة البيع"
   - امسح الباركود أو أدخله يدوياً
   - أضف العميل إذا كان مسجلاً
   - طبق الخصم إذا لزم الأمر
   - اختر طريقة الدفع وأتمم العملية

### نصائح للاستخدام

- **استخدام قارئ الباركود**: صل قارئ باركود USB وسيعمل مباشرة مع النظام
- **النسخ الاحتياطي**: قم بإنشاء نسخة احتياطية دورياً من "ملف" > "نسخة احتياطية"
- **تنظيم البيانات**: استخدم أسماء واضحة للمنتجات والموردين لسهولة البحث
- **تحديث المخزون**: راقب المنتجات منخفضة المخزون (المميزة باللون الأحمر)

## هيكل قاعدة البيانات

يستخدم النظام قاعدة بيانات SQLite مع الجداول التالية:

- **Products**: بيانات المنتجات والمخزون
- **Suppliers**: بيانات الموردين
- **Customers**: بيانات العملاء
- **Invoices**: الفواتير الرئيسية
- **InvoiceItems**: تفاصيل عناصر الفواتير
- **Returns**: المرتجعات (قيد التطوير)
- **Settings**: إعدادات النظام

## استكشاف الأخطاء

### مشاكل شائعة وحلولها

**المشكلة**: البرنامج لا يبدأ
- **الحل**: تأكد من تثبيت .NET 6.0 Runtime

**المشكلة**: لا يمكن حفظ البيانات
- **الحل**: تأكد من صلاحيات الكتابة في مجلد البرنامج

**المشكلة**: قارئ الباركود لا يعمل
- **الحل**: تأكد من تثبيت تعريفات قارئ الباركود وأنه معرف كـ HID device

**المشكلة**: البيانات مفقودة
- **الحل**: استعد من النسخة الاحتياطية عبر "ملف" > "استعادة"
