# نظام إدارة متجر الملابس - النسخة المتقدمة

## وصف المشروع
نظام شامل ومتقدم لإدارة متاجر الملابس مطور بلغة C# باستخدام Windows Forms وقاعدة بيانات SQLite. يتضمن جميع المميزات المطلوبة لإدارة متجر الملابس بطريقة احترافية مع نظام صلاحيات متكامل وأمان عالي.

## المميزات الرئيسية

### 1. نظام المستخدمين والصلاحيات 🔐
- **أنواع المستخدمين:**
  - مدير النظام: جميع الصلاحيات
  - مدير المتجر: معظم الصلاحيات عدا إدارة المستخدمين
  - كاشير: نقطة البيع والمرتجعات فقط
  - موظف: صلاحيات محدودة
- تشفير كلمات المرور
- تسجيل دخول آمن
- تتبع آخر دخول للمستخدمين
- إدارة حالة المستخدمين (نشط/غير نشط)

### 2. إدارة المنتجات المتقدمة 📦
- معلومات شاملة (اسم، وصف، مقاس، لون، مورد، أسعار، كمية)
- دعم الباركود مع بحث فوري
- رفع صور المنتجات
- تنبيهات ذكية للمخزون المنخفض
- تلوين المنتجات حسب حالة المخزون
- حساب تلقائي للربحية

### 3. نقطة البيع المحسّنة 💰
- واجهة ثنائية القسم (منتجات + فاتورة)
- مسح باركود متقدم مع دعم USB
- تعديل الكميات بسهولة
- خصومات ذكية (نسبة أو مبلغ ثابت)
- طرق دفع متعددة مع حساب الباقي
- طباعة فورية أو معاينة
- ربط العملاء بالفواتير

### 4. إدارة المرتجعات الشاملة 🔄
- **أنواع المرتجعات:**
  - استرداد نقدي
  - استبدال منتج
  - رصيد للعميل
- **حالات المرتجعات:**
  - في الانتظار
  - معتمد
  - مرفوض
  - مكتمل
- تحديث تلقائي للمخزون
- تتبع أسباب الإرجاع
- تقارير المرتجعات

### 5. لوحة التحكم التفاعلية 📊
- **إحصائيات مباشرة:**
  - مبيعات اليوم والشهر
  - عدد الفواتير
  - المنتجات منخفضة المخزون
  - إجمالي العملاء والمنتجات
- **رسوم بيانية:**
  - مبيعات الأسبوع
  - أفضل المنتجات مبيعاً
- تحديث تلقائي للبيانات

### 6. نظام التقارير المتقدم 📈
- **تقارير المبيعات:**
  - تقارير يومية وأسبوعية وشهرية
  - تحليل الأرباح
  - مقارنة الفترات
- **تقارير المنتجات:**
  - أداء المنتجات
  - المنتجات الأكثر مبيعاً
  - تقارير المخزون
- **تقارير العملاء:**
  - أفضل العملاء
  - تاريخ المشتريات
- تصدير التقارير إلى CSV

### 7. نظام الطباعة المتكامل 🖨️
- طباعة الفواتير مع معاينة
- إيصالات حرارية
- تخصيص تخطيط الطباعة
- معلومات المتجر التلقائية
- رسائل ترحيب مخصصة

### 8. سجل النشاط والأمان 🔍
- تتبع جميع العمليات
- سجل تفصيلي للتغييرات
- فلترة حسب المستخدم والتاريخ
- رصد محاولات الدخول
- حفظ القيم القديمة والجديدة

### 9. إدارة المتجر 🏪
- **إعدادات المتجر:**
  - اسم وعنوان المتجر
  - معلومات الاتصال
  - رسائل الترحيب
  - حد التنبيه للمخزون
  - معدل الضريبة
  - العملة
- **النسخ الاحتياطي:**
  - نسخ احتياطية تلقائية
  - استعادة البيانات
  - جدولة النسخ الاحتياطي

## متطلبات التشغيل
- **نظام التشغيل:** Windows 10 أو أحدث
- **إطار العمل:** .NET Framework 4.8 أو أحدث
- **الذاكرة:** 512 MB RAM كحد أدنى
- **التخزين:** 100 MB مساحة متاحة
- **الدقة:** 1024x768 كحد أدنى

## طريقة التشغيل
1. **التثبيت:**
   ```
   - استخرج جميع الملفات
   - تأكد من وجود .NET Framework
   - شغّل ClothingStoreManager.exe
   ```

2. **أول استخدام:**
   ```
   - اسم المستخدم: admin
   - كلمة المرور: admin123
   - قم بتغيير كلمة المرور فوراً
   ```

3. **الإعداد الأولي:**
   - أدخل معلومات المتجر
   - أضف الموردين
   - أدرج المنتجات
   - أنشئ حسابات المستخدمين

## الهيكل التقني

### قاعدة البيانات 🗄️
```
- Users (المستخدمين)
- Products (المنتجات) 
- Customers (العملاء)
- Suppliers (الموردين)
- Invoices (الفواتير)
- InvoiceItems (أصناف الفواتير)
- Returns (المرتجعات)
- ActivityLog (سجل النشاط)
- Reports (التقارير)
- Settings (الإعدادات)
```

### معمارية البرنامج 🏗️
```
├── Models/ (نماذج البيانات)
├── Services/ (طبقة الخدمات)
├── Forms/ (واجهات المستخدم)
├── DatabaseHelper.cs (إدارة قاعدة البيانات)
└── Program.cs (نقطة البداية)
```

## دليل الاستخدام السريع

### للمديرين:
1. **إدارة المستخدمين**: المستخدمين > إدارة المستخدمين
2. **مراقبة النشاط**: المستخدمين > سجل النشاط
3. **لوحة التحكم**: التقارير > لوحة التحكم
4. **الإعدادات**: ملف > إعدادات

### للكاشيرين:
1. **نقطة البيع**: المبيعات > نقطة البيع
2. **إدارة المرتجعات**: المبيعات > المرتجعات
3. **العملاء**: المبيعات > إدارة العملاء

### للموظفين:
1. **عرض المنتجات**: المخزون > إدارة المنتجات (عرض فقط)
2. **نقطة البيع**: المبيعات > نقطة البيع

## استكشاف الأخطاء

### مشاكل شائعة:
**المشكلة**: لا يمكن تسجيل الدخول
- **الحل**: تأكد من كتابة اسم المستخدم وكلمة المرور بشكل صحيح

**المشكلة**: المنتجات لا تظهر في نقطة البيع
- **الحل**: تأكد من وجود كمية متاحة للمنتج

**المشكلة**: الطباعة لا تعمل
- **الحل**: تأكد من تثبيت الطابعة وتعريفها في النظام
